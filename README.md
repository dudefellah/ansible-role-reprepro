dudefellah.reprepro
=========

Install and setup reprepro to help manage your Debian repository.

Requirements
------------

None.

Role Variables
--------------

Role variables are viewable in (defaults/main.yml)[defaults/main.yml]. All
values should have comments explaining their use.
